Nome do projeto: Parque Aquático
● Nome dos integrantes: Aline e Diego
● Objetivo do programa: O objetivo do programa é simular o funcionamento de um parque aquático, oferecendo difeerentes atrações, 
como piscinas de ondas, tobogãs e lanchonetes, proporcionando uma experiencia divertida e organizada.
● Descrição de cada opção do menu: No menu principal tem as atrações principais, indiferente da opção que o usuário escolher, 
irá cair no menu segundário, onde ele irá escolher a atração que deseja se divertir.
